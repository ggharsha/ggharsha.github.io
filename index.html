Projects

  • Disco

    Disco is a full stack clone of the popular live chat application Discord, built with React.js, Redux.js, Ruby on Rails, and numerous libraries and APIs. Users can register and create servers, channels, and direct messages in order to chat with one another. The website has fully functional websockets so that shared state can update in real time.

  • codeUps

    codeUps is a platform where coding tutors and students can connect with one another to collaborate and share knowledge. Tutors can upload samples of their work to advertise their services, while students can look for a mentor who's the perfect fit. The site features filtered search functionality and video hosting, all built with the MERN stack.

  • Bringing Home the Bacon

    This project is built with ES6 Vanilla JavaScript and the Canvas API in order to render objects within the site's HTML. Bringing Home the Bacon is a side-scrolling game with scaling difficulty in progressive levels, with an easily adjustable gameplay loop to account for future additions to the game.
